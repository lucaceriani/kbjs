## kb.js

https://cdn.jsdelivr.net/gh/lucaceriani/kbjs/kb.min.js

```js
kb({
  a: () => console.log('a'),
  b: () => console.log('b'),
  c: () => console.log('c'),
  'alt+shift+q': () => console.log('ctrl+shift+q'),
  'g u': () => console.log('g u'),
  'alt+p u': () => console.log('alt+p u'),
})
```
